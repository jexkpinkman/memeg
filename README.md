Jexk Not Dev — Portfolio

Portfolio personal developer dengan desain modern berbasis HTML, CSS, dan JavaScript vanilla. Mengusung konsep single-page website dengan background video cinematic, efek glassmorphism, serta animasi scroll yang smooth dan interaktif.

---

📁 Struktur Project

├── index.html    # Halaman utama
├── style.css     # Styling & responsive layout
└── script.js     # Interaksi dan animasi

---

✨ Fitur Utama

- 🎥 Video Background dengan efek blur elegan
- 🧊 Glassmorphism UI modern + transparan
- 🚀 Smooth Scroll Navigation dengan active indicator
- 🎬 Scroll Animation memakai Intersection Observer
- 📊 Animated Skill Bars
- 🗂️ Filter Project (All / Web / Mobile / Design)
- 📩 Contact Form dengan validasi input
- 📱 Responsive Design + hamburger menu mobile
- 🪐 Avatar Ring Spin & orbit animation pada section skills

---

⚙️ Customisasi

🎥 Ganti Video Background

Edit bagian "<source>" di "index.html":

<source src="YOUR_VIDEO_URL.mp4" type="video/mp4">

---

🎨 Ubah Warna Tema

Edit CSS variable pada ":root" di "style.css":

--primary: #6b8cae;
--dark: #0d1117;
--text: #e6edf3;

---

➕ Tambah Project

Duplicate ".project-card" pada section Projects lalu ubah:

- "data-category" → "web / mobile / design"
- Icon project
- Judul & deskripsi
- Tags teknologi

---

📞 Update Contact

Edit bagian ".contact-info" di "index.html".
